## Building the App

### Using Xcode

1. Open the project:
   ```bash
   open SystemMonitorApp.xcodeproj
   ```

2. Select your Mac as the target

3. Build and run (⌘R)

4. The app will appear in your menu bar with customizable display

### Using Command Line

```bash
xcodebuild -project SystemMonitorApp.xcodeproj -scheme SystemMonitorApp -configuration Release build
```

## Usage

### Menu Bar Display
1. Click the gear icon in the popover to access settings
2. Go to "Menu Bar Display" section
3. Choose your preferred display mode from 20 options
4. Select network unit (KB/s, MB/s, Mb/s, GB/s, Gb/s, or Auto)
5. Enable speed test for accurate network monitoring
6. Toggle network percentage display
7. Toggle download/upload display in CPU + Network All mode
8. Toggle detailed labels for CPU: 45%, NET: 30% format
9. Toggle compact mode for space-efficient display
10. Customize animation speed (0.5x - 2.0x)
11. Customize rotation interval if using rotation mode
12. Toggle icon and percentage display as desired
13. Menu bar updates in real-time based on your selection

### Speed Test
1. Go to the Dashboard tab
2. Find the Speed Test section
3. Click "Start Test" to begin continuous monitoring
4. View real-time download, upload, latency, and jitter speeds
5. Click "Stop Test" to pause monitoring
6. Enable "Enable Speed Test" in settings to use this data for menu bar display

### Health Tab
1. Click the heart icon to access the Health tab
2. View your system health score (0-100%)
3. Check for any detected issues
4. Read recommendations for improving system health
5. Health score is calculated based on battery health, temperature, memory pressure, and disk space

### Main App
1. Click the menu bar icon to open the monitoring panel
2. **Dashboard Tab**: View main gauges, historical charts, quick stats, speed test, and performance
3. **Details Tab**: Access detailed statistics for each component
4. **Processes Tab**: View top 8 processes by CPU and memory usage
5. **System Tab**: View complete system information including serial number and MAC address
6. **Health Tab**: View system health score, issues, and recommendations
7. Click the gear icon to access settings
8. Customize themes, refresh intervals, notifications, display options, and menu bar display
9. Toggle individual monitors on/off
10. Export data using the export button in settings

## Customization

### Menu Bar Display
Modify the display modes in `MenuBarDisplay.swift`:
```swift
enum MenuBarDisplayMode: String, CaseIterable {
    case iconOnly = "Icon Only"
    case cpu = "CPU Only"
    case smart = "Smart Mode"
    // ... add more modes
}
```

### Network Units
Modify the network units in `MenuBarDisplay.swift`:
```swift
enum NetworkUnit: String, CaseIterable {
    case auto = "Auto"
    case kbps = "KB/s"
    case mbps = "MB/s"
    case mbits = "Mb/s"
    case gbps = "GB/s"
    case gbits = "Gb/s"
}
```

### Colors
Modify the color palettes in `ThemeManager.swift`:
```swift
private let pastelColors: [ColorType: Color] = [
    .cpu: Color(red: 0.98, green: 0.82, blue: 0.85),
    .network: Color(red: 0.98, green: 0.72, blue: 0.75),
    // ... etc
]
```

### Refresh Intervals
Adjust default refresh intervals in each monitor's `init()` method by modifying the Timer interval.

## Speed Test Feature
